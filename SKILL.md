# 毛孩子世界杯表情包生成器

## 1. Overview

本 Skill 面向新手用户。用户只需要上传一张清晰的单只宠物照片，并输入一个内置支持的 2026 世界杯参赛国家名称，Skill 会生成 4 张静态宠物世界杯足球表情包。

生成结果应呈现同一只宠物，穿着受所选国家常见足球配色启发的虚构足球服，做可爱、拟人化、适合分享的足球动作。

## 2. Core Goal

核心目标是稳定生成一组 4 张静态方形宠物足球表情包：

- 保留上传宠物的可见特征。
- 使用同一个宠物身份。
- 使用所选国家的安全足球氛围和常见足球配色灵感。
- 使用固定内置文案池。
- 可以使用虚构、非官方的足球徽章增强足球贴纸识别度。
- 不复制真实国家队球衣、真实国家队官方队徽、足协标志、FIFA 标识、赛事标识、赞助商标志、品牌标志、球员姓名或球员号码。
- 不直接绘制真实国旗、国徽、国家机关徽记、地图、争议地图、军事符号或官方国家象征。
- 不加入政治、攻击、歧视或争议内容。

只有在当前 AI 平台支持直接生成图片时，才执行生成。若当前平台不能直接生成图片，只显示不支持平台提示，不输出图片提示词或替代模板。

不要承诺 100% 相似，只能尽量保留照片中可见的宠物特征。

## 3. Required Inputs

用户必须提供：

1. 一张清晰的单只常见宠物照片。
2. 一个来自内置 2026 世界杯参赛国家列表的国家名称。

支持的宠物包括但不限于：

- 猫
- 狗
- 兔子
- 仓鼠
- 鹦鹉
- 其他常见宠物

照片要求：

- 只有一只明确的主宠物。
- 宠物脸部或主要特征清楚。
- 能看到头部、身体轮廓或关键外观特征。
- 没有清晰可识别的人脸。
- 没有隐私信息。
- 没有严重模糊、严重遮挡或过暗光线。

## 4. User Flow

如果用户没有提供完整输入，发送以下新手友好提示：

```text
请上传一张清晰的宠物照片，并输入一个支持的 2026 世界杯参赛国家名称。

我会帮你生成 4 张静态宠物世界杯表情包。

示例国家：
巴西
法国
阿根廷
英格兰

当前版本不支持自定义文案，会使用内置安全文案池生成：
今天我首发 / 冲鸭！ / 进球啦！ / 稳住别慌
```

处理流程：

1. 检查是否有一张合格的单宠物照片。
2. 检查用户输入的国家是否在内置国家列表中。
3. 如果当前平台不能直接生成图片，显示不支持平台提示，不生成图片，也不输出图片提示词。
4. 如果照片或国家缺失，请补齐缺失项。
5. 如果照片不合格或请求不安全，简短拒绝并给出合规下一步。
6. 如果输入有效，生成 4 张静态方形宠物世界杯足球表情包。

## 6. Built-in Text Pool

只允许使用以下 4 条固定文案：

1. 今天我首发
2. 冲鸭！
3. 进球啦！
4. 稳住别慌

不要新增、翻译、改写、替换或自定义文案。用户要求自定义文案时，简短拒绝并继续使用以上内置文案池。

拒绝示例：

```text
抱歉，当前版本不支持自定义文案。我会继续使用内置安全文案池生成 4 张静态表情包。
```

## 7. Image Quality Check

生成前检查照片是否适合：

- 单只宠物清晰可见。
- 宠物主体明确。
- 主要外观特征可辨认。
- 没有清晰可识别的人脸。
- 没有隐私信息、证件、地址、电话、账号等敏感内容。
- 没有严重模糊、过暗、过曝、严重遮挡或无法识别的问题。

如果照片不合格，请简短说明原因，并请用户重新上传一张清晰的单宠物照片。

## 8. Pet Feature Extraction

生成时尽量保留以下可见特征：

- 物种
- 毛色或羽色
- 斑纹、斑点、条纹
- 耳朵形状
- 眼睛颜色和表情
- 脸型
- 身体轮廓
- 整体气质

不要承诺完全复刻。可以说“会尽量保留照片中清晰可见的特征”。

## 9. Football Outfit and Country Rules

服装规则：

- 使用虚构足球服。
- 只能说“受所选国家常见足球配色启发”。
- 可以使用安全的足球元素，如运动短袖、短裤、护腕、头带、围巾、足球鞋风格元素。
- 为了增强国家主题识别度，可以加入虚构的非官方足球徽章，但必须避免任何真实官方标识。
- 可以使用通用盾牌形徽章、简单非官方足球徽章，或足球图标、爪印、星形、条纹、简单几何形状等安全抽象符号。
- 可以在虚构徽章中使用中性国家名称文字或常见国家缩写，但不得把文字设计成官方标志、足协标志、FIFA 标识、赛事标识、品牌标志或赞助商标志。
- 徽章必须明显是虚构、非官方的。
- 不得使用真实国家队球衣。
- 不得复制真实球衣设计。
- 不得使用真实国家队官方队徽、足协标志、FIFA 标识、赛事标识、品牌标志、赞助商标志、球员姓名、球员号码、精确官方徽章布局或精确官方版型。
- 不得直接绘制真实国旗、国徽、国家机关徽记、地图、争议地图、军事符号或任何官方国家象征。

国家规则：

- 只使用国家名称、通用足球氛围、常见安全配色灵感和可爱运动动作。
- 只能使用抽象色块、通用运动配色、安全足球氛围，以及受所选国家常见足球颜色启发的虚构足球服。
- 不使用国家刻板印象、攻击性对比或贬损表达。
- 不制造国家之间的对立、嘲讽或挑衅。

## 10. Political Safety Rules

不得生成：

- 政治口号
- 军事符号
- 真实国旗
- 国徽
- 国家机关徽记
- 地图
- 争议地图
- 官方国家象征或国家标志性权力符号
- 侮辱国家或民族的内容
- 反国家内容
- 国家对立、攻击或仇恨表达
- 歧视、暴力、色情、违法内容

如果用户提出上述请求，请简短拒绝，并引导回“可爱宠物足球动作 + 虚构足球服 + 固定文案”的安全方向。

## 11. Unsupported Inputs and Refusal Rules

以下情况必须拒绝或重定向：

- 人类照片
- 儿童、名人、网红、自拍、多人合照
- 宠物照片中有清晰可识别的人脸
- 多只宠物且没有明确主宠物
- 非宠物图片
- 宠物严重模糊、过暗、严重遮挡或无法识别
- 自定义文案请求
- 不安全文案请求
- 复制真实国家队球衣的请求
- 真实国家队官方队徽、足协标志、FIFA 标识、赛事标识、品牌标志或赞助商标志请求
- 精确官方徽章布局或精确官方球衣设计请求
- 球员姓名、球员号码请求
- 模仿具体足球明星、公众人物、受保护角色、品牌或特定艺术家风格的请求
- 真实国旗、国徽、地图、争议地图、官方国家象征请求
- 政治口号、国家机关徽记、军事符号、国家侮辱、歧视、仇恨、暴力、色情、违法内容

拒绝应短、礼貌，并提供合规下一步。

通用拒绝模板：

```text
抱歉，这个请求我不能生成。可以改为：上传一张清晰的单宠物照片，并选择一个支持国家，我会生成 4 张安全的静态宠物世界杯表情包。
```

## 12. Generation Rules

有效请求必须生成正好 4 张方形 1:1 静态表情包：

1. 文案：今天我首发  
   动作：自信首发姿势，骄傲站立或坐着，准备上场。
2. 文案：冲鸭！  
   动作：带球向前跑，或充满能量地加油。
3. 文案：进球啦！  
   动作：庆祝进球，跳起、挥爪、挥翅或开心欢呼。
4. 文案：稳住别慌  
   动作：冷静队长或守门员式姿势，沉稳可靠。

每张图都必须满足：

- 同一只宠物身份。
- 虚构足球服，受所选国家常见足球配色启发。
- 可以出现虚构、非官方的足球徽章，例如通用盾牌形徽章、足球图标、爪印、星形、条纹或简单几何形状。
- 不复制真实球衣。
- 不出现真实国家队官方队徽、足协标志、FIFA 标识、赛事标识、品牌标志、赞助商标志、精确官方徽章布局、球员姓名或球员号码。
- 不出现真实国旗、国徽、国家机关徽记、地图、争议地图、军事符号或官方国家象征。
- 简单背景。
- 方形 1:1 构图。
- 适合微信表情收藏和小红书分享。
- 没有人类。
- 没有政治内容。
- 除固定文案外，不添加任何其他文案。
